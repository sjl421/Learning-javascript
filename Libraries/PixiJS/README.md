# PixiJS

## Install 
- [pixi.js -cdn](https://cdnjs.com/libraries/pixi.js/)

```html
<script src="https://cdnjs.cloudflare.com/ajax/libs/pixi.js/4.5.4/pixi.min.js"></script>
```

```bash
$ npm install pixi.js
```
